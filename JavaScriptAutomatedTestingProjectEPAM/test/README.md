How to run Navigation Tests:
    1. Clone repo:
        - terminal: 
            git clone <>
            cd <>
    2. Install dependenceis:
        - terminal:
            npm install
    3. Run tests:
        - terminal:
            npm run wdio
Generating reports(allure):
    1. Run Allure report:
        - terminal:
            npm run allure:generate
    2. Open report in browser:
        - terminal:
            allure open
